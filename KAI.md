Usando Rust + Tauri com Tailwindcss quero uma plataforma de pesquisa rápida com IA

**KAI** is a lightning-fast, hotkey-driven desktop AI helper overlay designed for KDE Plasma and Linux desktops. Built with **Tauri v2** and modern **Tailwindcss**, it delivers an instantaneous, ultra-customizable AI assistant experience

## ⚡ Key Features

- **Instant Desktop Overlay:** Press `Meta + X` to toggle showing/hiding the assistant overlay anywhere on your desktop.

- **Real-Time OpenAI Streaming** Word-by-word streaming responses from local OpenAI, OpenRouter, or LM Studio.

- **Markdown & Code Highlighting:** Instant Markdown parsing with 1-click code copy buttons.

- **Auto-Hide on Loss of Focus:** Automatically hides when you switch to another window or press `Escape`.

# Design:

- Dark mode and Light mode
- Rounded corners 20px 
- Shadows
- Black and White as primary colors
